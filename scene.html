(static) update(deltaTime)

this function is called every frame. You can move the player or do other stuff within this function You also get the deltatime wich is how much time (in seconds) is passed since the last frame
Parameters:
Name Type Description

Example

  if(scene.exists(playerUniqueIdentifier)){
    // moves the player 50 pixels per second
